Setting up MSys2 for building CodeBlocks.txt

Install MSys2 from http://msys2.github.io/

First update msys2 core components (if you have not done it yet): 
pacman -Sy
pacman -S bash pacman pacman-mirrors msys2-runtime

# You must exit out from MSYS2-shell, restart MSYS2-shell, then run below command, to complete rest of other components update:
pacman -Su

# Exit out of MSYS2-shell, restart MSYS2-shell, then you are ready to use MSYS2-shell.
# Note: for initial installation & setup, please also see this page http://sourceforge.net/p/msys2/wiki/MSYS2%20installation/
# Here -S means "sync" (Add). And -U means add or upgrade; it requires an URL-like option I have yet to learn

pacman --version # Pacman v4.2.1-313-gf91f-dirty - libalpm v9.0.1

# Remove programs that are NOT known to be needed
pacman -R wget                      # Not sure if this is needed
pacman -R unzip                     # Not sure if this is needed
pacman -R mingw-w64-winpthreads-git # Not sure if this is needed
pacman -R autogen                   # Not sure if this is needed
pacman -R mingw-w64-i686-hunspell
pacman -R winpty # console
pacman -R mingw-w64-i686-gcc-ada  
pacman -R mingw-w64-i686-gcc-fortran
pacman -R mingw-w64-i686-gcc-libgfortran  
pacman -R mingw-w64-i686-gcc-objc 
pacman -R mingw-w64-i686-gdb
pacman -R mingw-w64-i686-libmangle-git
pacman -R mingw-w64-i686-tools-git
pacman -R python2
pacman -R ruby

# Install programs that are known to be needed
pacman -S zip                       # Needed to zip files during CB build
pacman -S git                       # Needed to do Git Clone of my work.
